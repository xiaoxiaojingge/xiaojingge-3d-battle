# 需求分析

## 背景
极地大乱斗是英雄联盟中的一种5v5游戏模式，比赛双方在嚎哭深渊地图上展开战斗，这种模式非常的好玩，更何况后来出现的海克斯大乱斗模式，现在我自己有个摸鱼网站，是个web网站，我想在其中加入一个类似的5v5的对战模式的游戏，这样就可以和网友激情战斗，促进友谊

## 目标
游戏名：摸鱼大乱斗（暂定名），能在我目前的摸鱼网站中进行的5v5对战的小游戏

## 实现方案
Web网站小游戏方式呈现

### 目前需要实现的核心功能MVP

1. **核心玩法还原**
   实现5v5实时对战，战场为单线地图（类似嚎哭深渊），无野区、无回城补给，强调持续团战与快节奏对抗。
2. **随机性与公平性**
   - 英雄/角色采用全队随机或有限随机池机制，避免每局阵容固化。
3. **便捷组队与社交**
   - 自由加入蓝队或者红队，人满即可开始游戏
   - 内置简易聊天/表情系统，方便玩家互动（促进友谊）。
   - 可展示对局战绩、点赞队友/对手。
4. **轻量化与低门槛**
   - 单局时长控制在10–15分钟，适合摸鱼场景。
   - 无需复杂装备购买或发育，鼓励直接交火。
   - 网页端流畅运行，支持PC/移动端自适应。
5. **对战匹配机制**
   - 简单ELO或随机匹配，保证快速开局（排队≤30秒）。
   - 允许自定义房间：创建5v5房间，邀请链接分享。
6. **数据与激励**
   - 记录胜场、连胜、MVP等数据，提供排行榜（周/总）。
   - 完成对局可获取积分（此积分直接使用我摸鱼网站的用户积分系统即可），用于兑换头像、特效等皮肤化装饰（非数值付费）。
7. **技术目标**
   - 采用WebSocket或类似技术实现实时同步。
   - 支持断线重连，掉线期间由AI临时托管。
   - 服务器资源轻量，支持同时在线100人以上。



### 拓展功能（后续实现）

#### 海克斯特色（进阶目标）

加入“海克斯强化”系统（每局随机全局增益效果），增加对局变数与趣味性（拓展功能），每局游戏开始前随机生成1–2个全局Buff（如：全体技能冷却-20%、全体移动速度+15%、死亡后原地自爆等，更多玩法等），并在界面显著提示，增加战术趣味。
